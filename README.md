# ✦ Celestara — Imaginary Luxury Emporium

A fancy, fictional boutique for exploring design. Dark cosmic theme, gold accents, animated starfield, and a **Curator Studio** to add products in seconds.

## Quick start

Open `index.html` in a browser, or run a local server:

```bash
npx serve .
```

## Add products (easy)

1. Open **Curator Studio** (`admin.html`)
2. Fill in name, description, price, category
3. Drop an image or paste an image URL
4. Click **Add to vault**

Products save in your browser automatically. To persist on GitHub:

1. Click **Download products.json**
2. Replace `data/products.json` in this repo
3. Commit and push

## Deploy on GitHub Pages

1. Push this folder to a GitHub repo
2. **Settings → Pages →** Source: `main` branch, folder `/` (root) or `/docs` if you move files
3. Your site will be live at `https://YOUR_USERNAME.github.io/REPO_NAME/`
